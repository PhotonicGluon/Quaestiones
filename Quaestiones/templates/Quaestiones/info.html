--- Frequently Asked Questions ---

What does the word Quaestiones mean? Quaestiones means questions in Latin. I felt that this name was appropriate as this is a questions-and-answers website, but aimed at a computer science crowd. The transmission of Greek logic to the West was also done through the latin medium, so it is also poetic in this way.

Help! I'm stuck on a question! If you get stuck, try your solution against the example(s) given in the question. You should get the same answers. If you do not, re-read the description of the question. Did you misunderstand something? If the examples work, and your answer is still incorrect, build some test cases for which you can verify the answer by hand and see if those work with your program. Make sure you have the entire question's input. If you're still stuck, you could ask a friend for help, or come back to the question later.
